Ideas
Food data-> interactive visualization about the components of popular [food type] items. Data on ingredients + source of ingredients.
Vis tpye


[food type] = snacks, fast food, sugary drinks, specific junk foods i.e. chips, chocolate bars, etc.

Task 1:
Modify the bubble chart to allow clicking on a bubble to reveal more information by applying additional filters to the bubble chart.
- When the first dropdown is set to nutriscore_grade or ecoscore_grade: clicking on a bubble filters data for data points with the grade of the bubble clicked on, 
  then reveals the bubble chart using frequency of brands as the values.
- When the first dropdown is set to brands: clicking a bubble reveals a second dropdown bar. This second dropdown bar has two options: ecoscore_grade
  and nutriscore_grade. We filter data for points with the brand of the bubble clicked on, then apply the selection of the second dropdown bar 
  either nutriscore_grade or ecoscore_grade values, to the bubble chart.
- Finally, after we have clicked on a bubble, a back button should appear in the top left. clicking it should take us to the bubble chart based solely
  on the first dropdown bar (without any additional filtering)

Task 2:
Modify the position of the dropdown menus and back buttons.
- The first dropdown menu, that handles initial filtering, should be above the svg and centered in the screen.
- When clicking on a bubble: 
    The back button should appear above the svg to the left. That is, the whole of it should be above the svg, and its left edge should line up with the left edge of the svg.
    The second dropdown menu, if it appears (because the first dropdown is set to Brands) should appear above and to the right (above, with its right edge lining up with the right edge of the svg)

Task 3:
- When the initial filtering is one ecoscore_grade or nutriscore_grade, or when we drilled down on a brand, the colours should match the
  official colour scheme for these grades: nutriscore_grade's colour scheme is displayed in nutriscore.jpg open as an editor currently.
  It ranges from A:dark green, B:light green, C:yellow, D:orange, E:red. ecoscore_grade uses the same colour scheme.
- When drilling down on a bubble, the tooltip should stop displaying "Click to see breakdown."

Task 4:
- Leave out the not applicable and not available data points from the bubble chart. 
- Change the ecoscore_grade colour scheme slightly:
    A+:dark green
    A:green
    B:light green
    C:yellow
    D:light orange
    E:orange
    F:red
- Add a side panel for a legend. The legend should update whenever a selection is made (either an initial selection, drilling down, or backing out)

Task 5:
- for ecoscore_grade, ensure that A-PLUS in the bubble chart matches the A+ entry in the legend and colour scheme. A-PLUS in the bubble chart should be dark green and be labelled A+.
- for the legend when viewing brands or drilling down to see brands: change the legend entries to match the names in the bubble chart.
  In particular, the capitalization should be as normal for a proper noun.
